Meet-O-Doctor

An application that allows people to join one-on-one online session with a doctor as well as book for a face-to-face session with a doctor.

There are numerous amount of unnecessary doctor visits through out every year causing problem such as long wait time.

This application aims to reduce those unnecessary visits by suggesting people to join an online session with a doctor instead of visiting a clinic/hospital.

The system would use a queueing system concept in which there would be a limit in the number of people queueing for a doctor. This would cause people to psychologically think the wait time is shorter than expected.

This is only the design concept of the application, there are no functionalities applied yet.

Project Description

Project Aim

As an application that provide medical services, Meet-O-Doctor aims to provide a fast and convenient services for people. An app that has a short queueing time to reduce people's annoyance to long queue lines in hospitals.

A three-in-one functionality app that allows you to seek a doctor, buy medicine and read medical news updates
